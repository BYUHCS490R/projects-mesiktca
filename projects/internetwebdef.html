Internet

The internet is one big global network that allows users from all the world to communicate and share information when connected.

Web

The web is service that retrieves information and gives users access to webpages in the internet. The web can be thought of as a collection of hyperlinks. If a user clinks on one item, the user is connected to another item or page. This is the service that the Web offers.

How are the Internet and Web different?

The Internet is the whole global network that connects users via computers around the world. Users then can share information like websites. The Web is a service on the Internet that gives users access to the shared information like websites.

ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange. The ASCII allows computers to communicate in code while users write commands using English letters and character. Each letter and character is assigned to a numberical value.

Unicode is a universal code system. Think ASCII, but global. Each character, letter, and symbol from every known language is given it's own unique number in Unicode. Each unique number is called a "code point."

What is UTF-8

UTF-8 takes the code points from Unicode and translates into binary numbers that computers can use and store.

URL

URL stands for Uniform Resource Locator. A URL is a specific and unique address used on the web that brings you to a specific resource such as a website.

Domain name

The domain name is a unique and easy-to-remember name/indentifier that users can type into their browers to pull up a specific website. Example: google.com

HTTP

HTTP stands for HyperText Transfer Protocol. It is the protocol or rules that tells computers how to send and receive information for websites.

HTTPS

HTTPS stands for Hypertext Transfer Protocol Secure. This is the secure versiono of HTTP. When information is being sent, HTTPS will encrypt the information or data so that it is secure while on network.

IP Address

Each computer or device that connects to a network or Internet is given a specific, unique numerical label called the IP address. The IP address allows your device to be located while on a network.

DNS

DNS stands for Domain Name System. The system takes domain names like facebook.com and turns it into an IP address. The IP address then allows communication between computers. This allows Facebook website to be delivered and load into the right computer.

W3C Recommendations

The W3C Recommendations, World Wide Web Consortium, are a set of rules on how a website should work so that websites are consisent and accessible.

XML

XML stands for Extensible Markup Language. It is a way for users to describe the contents and the layout or structure.

XHTML

XHTML stands for Extensible HyperText Markup Language.It is similar to an HTML but it has the rules of XML. It is more strict than HTML.

HTML

HTML stands for HyperText Markup Language. It's the coding language that allows users to create websites.

HTML5

HTML5 is the most updated version of HTML.
