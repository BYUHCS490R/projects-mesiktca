What kind of photos are good to encode using JPEG?

JPEG is good for photos with lots of colors and details.

What kind of photos are good to encode using GIF?

GIF are best for simple photos like logos, icons, small moving images

What is unique about the PNG format?

It doesn't lose quality and supports transparency. It is best for when clarity matters.

How do you resize your photo to the size it should be displayed?

Download the image, open image in photo editor, resize button- resize image

Why should you resize photos using photo editing software instead of resizing it using CSS?

If the file is done through CSS, then when the web loads the image will still be downloaded at hits full size and then resized which wastes data and bandwith.
