Different ways to select an item One way is by tag name. By selecting tag names this allows the items in those tags across the entire page will be uniform. Another way is the descendant rule: example selecting header tag and then the H1 tag. Another way is to give it an ID.

ID vs Class ID = # Class = . With an ID, only one item on the page can have the unique name. ID names the item. Classes can be reused. More than one class can be given to an item. Classes should be named using adjectives eg. purple, flashy, smaller.

Span vs DIV Span tag is a way to select an item inline. Span tag doesn't do anything on its own unless selected. DIV tag is a way to select and group things together. DIV tag will cause line breaks. Span won't.
