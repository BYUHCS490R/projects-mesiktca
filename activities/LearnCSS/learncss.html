Explain the difference between inline, embedded, and external CSS. Inline is when you use style attributes in the HTML- it styles a specific element. Embedded are usually style tags found in the header that apply to the whole page. CSS- the styling is kept in a seperate file that can be linked via link in the header.

Explain why you should only use external CSS. We should only use external CSS because it's easier organizationally- you don't have to sort through code and styling. One stylsheet can be used in multiple pages- this always your pages to have a consistent look. If you need to update your pages look, you only have to worry about changing one file.

What is a selector used for? Selectors allow users to pick a specific part of the page that they want to style. It targets specific elements.
